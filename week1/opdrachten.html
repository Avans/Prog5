Elke week zijn er een aantal makkelijke, medium en moeilijke opdrachten waar je in het practicum aan gaat werken. Aan het begin kan je de uitwerking van de opdrachten aan je docent laten zien. Deze zal ze afvinken en je zult hiervoor avans punten krijgen.

Voor het huiswerk van elke weeg staat er een start project klaar. Deze startprojecten zijn onderdeel van de github repository waarmee dit webdictaat gemaakt is. Om deze start projecten te gebruiken kun je het beste de github repo clonen. Rechts van dit informatie blok staat een link en een clone button. Voor elk project is meteen een uitwerking toegevoegd. Gebruik deze om te spieken als je er even niet meer uit komt.


Daarnaast bevat elke opdracht, ook een Unit test project en een aantal interface objecten. Gebruik deze interfaces om je eigen objecten te immplementeren, en run de tests om te controlleren of de opdracht goed is.

De workshop opdrachten zijn tevens huiswerk opdrachten. Deze opdrachten zijn oefenmateriaal voor het assessment aan het einde van dit blok. De opdrachten zijn niet verplicht, maar de eindopdracht wordt pittig. Ik kan je aanraden deze opdrachten dus netjes bij te houden.

Opdracht - "Hello World"

In deze opdracht ga je een Console Applicatie maken in het .NET Framework met behulp van Visual Studio en C#. C# lijkt qua syntax heel erg op Java dus het is niet erg als je nog nooit eerder iets met C# gedaan hebt, daar waar nodig worden hints gegeven.

Een nieuwe taal leren? Dan eerst hello world! We gaan het alleen iets moeilijker maken. Bouw een console applicatie waar de console vraagt om een naam. Nadat de gebruiker een naam heeft ingevoerd moet de console "hello" + naam printen.

  • Let op dat je een template kiest met de taal C#.
  • Let op dat de template die je kiest een console application template is.

Opdracht - Leeftijd calculator in .NET

Voor deze opdracht is er een startup project beschikbaar. In dit project zitten interfaces waar je gebruik van kan maken en test die je kan runnen. Het is niet verplicht om gebruik te maken van deze startup, dus je kan ook zelf een nieuw project beginnen.

Het doel van deze oefening is een eerste kennismaking met C# en het leren omgaan met Visual Studio. Maak in Visual Studio de Console Application template een C# programma die een geboortedatum van een persoon vraagt en vervolgens de leeftijd van deze persoon berekent en toont. De applicatie dient aan de volgende voorwaarden te voldoen:

  • De datum dient via de Console opgevraagd te worden en aan het format (dd-mm-yyyy) te voldoen.
  • Als een ongeldige datum of ongeldige input gegeven wordt dient dit op de Console aangegeven te worden.
  • Bij een geldige datum wordt de leeftijd berekend en getoond.
