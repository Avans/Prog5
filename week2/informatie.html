Properties

Een klasse heeft over het algemeen private en public fields. Private fields kunnen alleen benaderd worden vanuit de klasse zelf. Public fields zijn ook toegankelijk van buitenaf.

Als je in java een field alleen leesbaar wil maken dan is het gebruikelijk om een private field te maken met een bijbehorende functie als 'getter'. Daarnaast is het mogelijk een functie als 'setter' toe te voegen en eventuele validatie uit te voeren op de nieuwe waarde van het veld.

In C# doen we dit net even anders. Hier hebben we namelijk een speciale syntax voor het maken van getters en setters. Dit zijn properties. Je bent bijna verplicht om in C# altijd Properties te gebruiken in plaats van normale 'fields'. Sommige frameworks (waar we in de toekomst gebruik van gaan maken) werken bijvoorbeeld alleen goed samen met properties en niet met fields.

Java

private String myField; 

public String getMyField()
{
    return this.myField;
}
public void setMyField(String value)
{
     this.myField = value;
}

C#

public class Person
{
    private string _firstname;

    public string Firstname
    {
        get
        {
            return _firstname;
        }
        set
        {
            _firstname = value;
        }
    }
}

Dit kan ook in een kortere variant. het onderstaande voorbeeld zal er onderwater een private field gemaakt worden waarmee de getter en setter hun waarde benaderen. Het is slim om jezelf aan te leren deze korte syntax te gebruiken voor properties i.p.v public fields. Hiermee kun je in de toekomst makkelijker bijvoorbeeld de 'set' private maken.

Daarnaast is het ook mogelijk om zelf te bepalen of je een get of een set implementeerd. De implementatie mag ook meet logica bevatten.

Korte syntax

public class Person
{
    private string _firstname;

    //De lange variant (ingeklapt)
    public string Firstname...
	
    //De korte variant
    public string Lastname { get; set; }
	
}

Logica

public class Person
{
    private string _firstname;

    //De lange variant (ingeklapt)
    public string Firstname..

    public string Lastname { get; set; }

    public string Fullname 
    { 
        get
        {
            return _firstname + Lastname;
        }
    }
}
