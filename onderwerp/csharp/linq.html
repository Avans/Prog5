Nu gaan we gebruik maken van Linq. Linq is een assembly (system.linq) waarmee je filter-functies kunt uitvoeren op lijsten. Deze functies kun je uitvoeren doormiddel van een linq query of een lambda expressies. 

	using System.Linq;

Deze Linq query's maken je leven een stuk makkelijker! Deze voorbeelden zijn maar het topje van de ijsberg wat je allemaal kan met Linq, en ik kan je aanraden om ook even op het wereld wijde web verder te zoeken!

Let op! De return value van een linq statement is van het type IEnumerable. Dit is een ander type dan List. Wil je toch een List als result?

Linq query & Linq met lambda

 
//linq query
return from p in People where p.FirstName[0] == firstChar select p;

//linq met lambda
return People.Where(p => p.FirstName[0] == firstChar);
