Visual Studio

Programmeren met een nieuwe taal vereist een nieuwe werkomgeving. Waar je met java je applicaties in netbeans of eclipse schreef, gaan we nu aan de slag met Visual Studio. Deze tool is gemaakt door Microsoft zelf en ondersteunt alle talen waarmee je .NET applicaties kan schrijven.

Wij gaan alleen applicaties schrijven in C#, maar het is ook mogelijk om applicaties te bouwen in C++ of Visual Basic.

Er zijn verschillende versies van Visual Studio beschikbaar. Wij gaan gebruik maken van de gratis versie. Visual Studio Community

Je kunt Visual Studio Community gratis downloaden via de website van visual studio

https://www.visualstudio.com/

Andere versies van Visual Studio zijn normaal niet gratis. Maar omdat Avans toegang heeft tot MSDNAA kunnen studenten hier ook de andere versies gratis downloaden.

https://msdnaa.avans.nl/
