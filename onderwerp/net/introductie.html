Welkom in de wondere wereld van .NET.

In deze eerste les gaan we onze eerste applicaties schrijven in de taal C# (spreek uit: sie sjarp). Om dit te doen gaan we gebruik maken van de IDE* Visual Studio. De taal C# lijkt in syntax enorm veel op Java. Veel concepten zullen je dus ook bekend voorkomen.

Daarnaast gaan we kijken naar een aantal nieuwe concepten. Termen als Linq, Lambda, Methods, Properties en Delegates zullen de komende weken uitgelegd gaan worden.

Aan het einde van dit blok heb je alle skills in huis om een Windows 8 applicatie te bouwen. Dit wordt ook de eindopdracht.

*IDE - Integrated development environment

Java vs .NET

De wereld van .NET verschilt nogal met die van Java. Om te beginnen is .NET platform afhankelijk. Applicaties geschreven in C# werken alleen op Windows machines. Daarnaast wordt er binnen .NET veelvuldig gebruik gemaakt van assemblies. Dit zijn pakketjes met klassen waar jij als ontwikkelaar gebruik van kan maken.

Elke Windows machine heeft een standaard set aan assemblies waar je als ontwikkelaar meteen bij kan. Daarnaast zijn er eindeloos veel assemblies die je kan "include-en" in je project. Verderop deze pagina gaan we meer aandacht besteden aan het onderwerp assemblies.

Hieronder zie je een overzicht van de onderdelen van het Java framework ten opzichte van het .net framework. Kijk eens goed naar de verschillen.

Maak je geen zorgen als een aantal termen nog onbekend voorkomen. We gaan alle onderdelen de komende weken behandelen. Er zijn een aantal belangrijke verschillen.

  • Java is platform onafhankelijk, .Net draait alleen op windows.
  • Java maakt gebruik van 1 grote JDK (Java Development Kit), .NET heeft verschillende kleinere assemblies.
  • Java heeft een Virtual Machine, .NET gebruikt CLR (Common Language Runtime).
